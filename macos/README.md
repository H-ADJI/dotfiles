# macOS Setup

First clean-install goal: get enough tools to continue building dotfiles comfortably.

Installs:

- Lix
- Homebrew
- Google Chrome
- Ghostty
- OpenCode

Run:

```bash
cd ~/dotfiles
bash macos/setup/main.sh
```

After bootstrap:

```bash
open -a Ghostty
opencode
```

Then continue with nix-darwin + Home Manager setup.
